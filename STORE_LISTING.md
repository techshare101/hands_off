## Long Description
HandOff transforms your browser into a dynamic AI command center. Built on Manifest V3, it features a modular architecture that connects AI models, MCP servers, and A2A agents without leaving your workflow.

Key Features:
- Connect Hub: Search, enable, and manage 13+ pre-configured AI apps and tools with zero-code setup.
- Dynamic Routing: Hot-swappable tool registry that syncs instantly with your decision router.
- Multi-LLM Support: Route prompts across Gemini, OpenAI, Anthropic, Groq, DeepSeek, Qwen, Mistral, OpenRouter, and RouteLLM.
- MV3 Optimized: Lightweight service worker with intelligent keep-alive, context pruning, and circuit breaking.
- Telemetry Dashboard: Real-time system health, tool usage metrics, and error tracking — 100% local, zero external tracking.
- Composio Integration: Connect to 250+ apps through Composio's unified tool platform.
- A2A Protocol: Delegate tasks to remote AI agents via the Agent-to-Agent protocol.
- MCP Client & Server: Full bidirectional MCP support for tool discovery and execution.

Privacy & Security:
- All data stays in chrome.storage.local.
- No external telemetry, analytics, or tracking.
- You control every API key, endpoint, and connection.
- LLM API calls go directly to your configured provider. HandOff never proxies or stores them.

Built for Developers & Power Users:
HandOff bridges the gap between browser automation and agentic AI. Whether you're orchestrating multi-step workflows, testing MCP servers, or routing LLM calls dynamically, HandOff provides the infrastructure to scale your AI toolkit.

Ready to upgrade your browser into an AI agent? Install HandOff and take control.
